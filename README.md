# YouTube & TikTok Always Play

A Chrome extension that prevents videos from automatically pausing when you switch tabs.

## Installation
1. Download this repository as a ZIP archive and extract it into a folder.
2. In Chrome, navigate to `chrome://extensions/`.
3. Enable **"Developer mode"** (toggle switch in the top-right corner).
4. Click the **"Load unpacked"** button and select the extracted folder.

# YouTube & TikTok Always Play

Расширение для Chrome, которое отключает принудительную паузу видео при переключении вкладок.

## Установка
1. Скачайте этот архив и распакуйте в папку.
2. В Chrome перейдите на страницу `chrome://extensions/`.
3. Включите **«Режим разработчика»** (Developer mode).
4. Нажмите **«Загрузить распакованное расширение»** и выберите созданную папку.
